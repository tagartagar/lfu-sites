Экономика





Экономика - система хозяйствования, обеспечивающая общество необходимыми благами, наука о ведении хозяйства.
  • Макроэкономика;
  • Микроэкономика.
Факторы производства - ресурсы, непосредственно задействованные в производстве.
  • Труд: факторный доход - зарплата;
  • Земля: факторный доход - рента;
  • Капитал: факторный доход - процент;
  • Предпринимательская способность: факторный доход - прибыль;
  • Информация: факторный доход - роялти.
Экономическая система - система организации хозяйствования, обеспечивающая общество необходимыми благами.
  • Традиционная;
    • Коллективная форма собственности;
    • Натуральная форма хозяйствования;
    • Бизнес не развит.
  • Командная;
    • Государственная форма собственности;
    • Директивное ценообразование;
    • Уравнительное распределение.
  • Рыночная;
    • Частная форма собственности;
    • Развитая конкуренция;
    • Свобода предпринимательства.
  • Смешанная = рыночная + командная.
  • Преимущества рынка:
    • Эффективное распределение ресурсов;
    • Стимулирует научно-технический прогресс;
    • Развитая конкуренция;
    • Отсутствие дефицита;
    • Свобода предпринимательства.
  • Недостатки рынка:
    • Цикличность развития;
    • Не производит общественные блага;
    • Усугубляет экологические проблемы;
    • Склонен к монополизации;
    • Не может справиться с бедностью и безработицей.
  • Преимущества плановой экономики:
    • Отсутствие безработицы;
    • Низкая инфляция;
    • Нет кризисов перепроизводства;
    • Стабильность развития;
    • Не испыпытывает циклических колебаний.
  • Недостатки плановой экономики:
    • Дефицит товаров;
    • Отсутствие свободы предпринимательства;
    • Нет стимулов к труду;
    Собственность - принадлежность вещей определенным лицам.
    • Частная;
      • Собственность физ. лиц;
      • Собственность юр. лиц.
    • Государственная;
      • Собственность субъектов;
      • Федеральная собственность.
    • Муниципальная.



    Рынок - система хозяйствования, обеспечивающая общество необходимыми благами.

    • По уровню: мировой, региональный, национальный, местный;
    • По объекту купли-продажи:товаров, услуг;
    • По законности:легальный, нелегальный;
    • По типу конкуренции:рынок совершенной конкуренции, рынок несовершенной конкуренции;
    • По характеру продаж:оптовый, розничный;
    • По уровню насыщенности:избыточный, равновесный, дефицитный;
    • По географическому признаку:внутренний, внешний.
  • Функции рынка:
    • Посредническая;
    • Регулирующая;
    • Ценообразующая;
    • Санирующая;
    • Информационная;
  • Для рынка характерны:
    • Неконтролируемый спрос;
    • Неконтролируемое предложение;
    • Неконтролируемая цена;

    Политика — 1) целенаправленная деятельность людей, направленная на получение и удержание власти. 2) события общественной жизни, связанные с деятельностью государства, органов государственной власти, государственным управлением.
  • Субъекты политики:
    • Избиратели;
    • Политические партии;
    • Общественные движения;
    • СМИ;
    • Государственные деятели;
    Власть — способность влиять на другого человека, независимо от его желания.
  • Структура власти:
    • Субъект;
    • Объект
  • Виды ресурсов власти:
    • Силовой;
    • Финансовый;
    • Информационный;
  • Три ветви власти:
    • Законодательная;
    • Исполнительная;
    • Судебная;
